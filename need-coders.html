09-Jan-2014 0.11.1

We Need Coders Passionate About Quality

ReXSL Java web framework is a RESTful integration of JAX-RS, JAXB and XSLT. Using ReXSL you develop a RESTful API that automatically, in runtime, turns into an HTML web-site by means of XSLT.

We started to develop ReXSL in August 2011 and it is already being successfully used by a few open source (e.g. s3auth.com) and commercial (e.g. expinia.com) projects.

We need your help with coding, in order to release the first stable version (we’re still in 0.x state). If you’re passionate about quality of code and is interested to contribute or learn something new, email me: yegor@tpc2.com.
