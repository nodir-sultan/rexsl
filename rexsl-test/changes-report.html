Release History

Version Date Description

Release 0.4.12 - 7 May 2013

Type Changes By
XhtmlMatchers understand InputStream and Reader, converts them to String and uses as XML Fixes 691. yegor

Release 0.4.4 - 19 Nov 2012

Type Changes By
Class com.rexsl.test.RestTester automatically adds User-Agent HTTP header to all outbound HTTP requests, when such a header is not specified by the client. The header contains ReXSL version and JVM version. Fixes 612. yegor

Release 0.4.3 - 18 Nov 2012

Type Changes By
Class com.rexsl.test.html.NoBrokenLinks was making RESTful requests to all URLs, which lead to problems with image and other non-Unicode resources. The problem is fixed by a usage of a traditional HttpURLConnection calls. Fixes 610. yegor
