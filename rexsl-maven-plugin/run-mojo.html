rexsl:run

Full name:

com.rexsl:rexsl-maven-plugin:0.13:run

Description:

Run WAR as a web server to enable interactive testing of a web application.

Run mvn clean package rexsl:run -Drexsl.port=9099 and open your browser at http://localhost:9099/. You will see the front page of your application. What is especially interesting is that you can make changes to static resources (xsl, css and other files in src/main/webapp) and then just refresh the page in a browser. New versions of the files will be delivered. That's how you can interactively test your XSL stylesheets.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.

Parameters

(no parameters)
